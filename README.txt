== Description ==

This plugin will remove the unwanted p tag from the content.


== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Remove Unwanted P tag from content'
3. Activate Remove Unwanted P tag from content from your Plugins page.

= From WordPress.org =

1. Download Remove Unwanted P tag from content.
2. Upload the 'remove-unwanted-p-tag-from-content' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Remove Unwanted P tag from content from your Plugins page.
